## 🚀 Como começar (5 minutos)

1. **Stripe**: crie product + price (R$ 297). Pegue as 4 chaves.
2. **.env**: copie `.env.example` para `.env` e preencha.
3. **Teste local**: `npm install && npm start`
4. **Deploy**: `vercel --prod`
5. **Webhook**: configure no Stripe apontando para `/api/webhook`
6. **Links**: use a landing page para vender.

Pronto! Começa a vender.
